# 网络信息

Surge 面板脚本，用于显示本地、入口、落地三段 IP 信息。当前版本不再依赖外部映射库。

## 文件

- `网络信息.sgmodule`：Surge 模块入口。
- `网络信息.js`：面板脚本主体。

## 优化点

- 直连、本地、代理数据源互相独立，弱网下自动降级。
- 无需下载 `映射库.json`，首次运行不再依赖 GitHub 字典连通性。
- 智能并发查询：首个数据源成功后只短暂等待更优字段，避免慢接口拖住面板。
- 位置优先使用数据源中文字段，国家名由 JS 原生 `Intl.DisplayNames` 智能生成。
- 多源字段融合，中文位置、运营商、ASN 会从不同数据源择优合并。
- 面板顶部显示完整路线：本地 -> 入口 -> 落地，并为每段自动加国旗。
- 每段显示 IP、位置、运营商、ASN、国家/区域/城市和实时/缓存状态。
- 自动标记同国/跨境、直落/中转，并显示本次刷新耗时。
- 入口 IP 变化会触发刷新，同一节点短时间内直接走缓存。
- 本地、入口、落地支持分段缓存继承，单个数据源失败时不清空整块面板。
- 面板采用三行一组的紧凑排版，信息更多但仍然容易扫读。
- 查询、缓存状态、渲染分层处理，后续维护和扩展更清晰。
- 地址、运营商、ASN 做统一清洗，输出更稳定。

## 架构

脚本按 `配置 -> 请求 -> 智能格式化 -> 查询 -> 入口识别 -> 缓存状态 -> UI 渲染` 分层组织。

缓存中保存结构化数据；刷新时优先使用新查询结果，单段失败时只继承对应旧段，并在 UI 中标记 `缓存`。

## 使用

在 Surge 中安装 `网络信息.sgmodule`，面板会每 5 秒触发一次脚本；脚本内部有 30 秒结果缓存，节点切换时会尽快刷新。

模块安装链接：

```text
https://raw.githubusercontent.com/justin19933/waibucangku/main/%E7%BD%91%E7%BB%9C%E4%BF%A1%E6%81%AF.sgmodule
```

Surge 一键安装：

```text
surge:///install-module?url=https%3A%2F%2Fraw.githubusercontent.com%2Fjustin19933%2Fwaibucangku%2Fmain%2F%25E7%25BD%2591%25E7%25BB%259C%25E4%25BF%25A1%25E6%2581%25AF.sgmodule
```
